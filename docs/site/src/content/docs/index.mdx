Ironbullet is a desktop application for constructing, inspecting, and testing visual execution pipelines. A pipeline is made from blocks for request construction, parsing, checks, transformations, and controlled flow.

## What the application does

- Builds ordered pipelines with a visual editor.
- Runs local debug executions with response and variable inspection.
- Imports supported OpenBullet-compatible configuration formats.
- Exports supported pipelines to Rust source.

## Start with the supported path

1. Create a pipeline.
2. Add an **HTTP Request** block.
3. Add a **Parse** block to extract response data.
4. Add a **Key Check** to classify explicit conditions.
5. Run against an authorized local or test endpoint.

See [Capability status](/Ironbullet/capability-status/) before relying on an advanced block in a production configuration.
