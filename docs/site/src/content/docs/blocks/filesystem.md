# File system

Use local file operations only against paths that are part of your authorized workspace.

All entries use the selected block's settings panel. The focused configuration guidance below describes the fields to review before using the block.

## File System

- **Rust type:** `FileSystem`
- **Availability:** Palette
- **Purpose:** Performs the selected file-system operation using the configured paths and variables.
- **Configure:** Set the operation and paths explicitly. Use variables for workspace-local paths rather than embedding environment-specific locations.
