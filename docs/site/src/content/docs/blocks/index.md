# Block reference

This library covers every `BlockType` declared by the native pipeline model. Entries are organized by the same category used by the application.

## How to use an entry

- **Palette** means the block can be added from the current Block Palette.
- **Compatibility-only** means the block remains readable for imported configurations but is not offered for new pipelines in the palette.
- **Unavailable in native engine** means the type is preserved for configuration compatibility, but attempting to execute it reports an explicit engine error.

Use the [Interface tour](/Ironbullet/interface-tour/) to locate the palette, canvas, settings panel, and debugger. Use [Capability status](/Ironbullet/capability-status/) before building a workflow around an advanced integration.
